## Overview

**StatSphere** is a centralized sports statistics database that stores and manages detailed data for multiple professional leagues, including the NFL, NBA, MLB, and NHL. It enables users to query information about teams, players, games, and seasons through a simple command-line interface.

## Features

- **Multi-League Support**: Includes major leagues like NFL, NBA, MLB, and NHL.
- **Comprehensive Data**: Stores information on teams, players, games, and seasons.
- **User-Friendly CLI**: Easy-to-use command-line interface for data queries.
- **Data Integrity**: Utilizes relational database principles to ensure consistent data.

## Installation

### Prerequisites

- **Python 3**: [Download Python](https://www.python.org/downloads/)
- **SQLite3**: Most systems have SQLite installed. If not, download from [SQLite Download Page](https://www.sqlite.org/download.html).

### Setup Steps


How to Run:
1. Ensure you have Python 3 installed.
2. Ensure you have SQLite3 installed.
3. In a terminal, navigate to the directory containing schema.sql, data.sql, and app.py.
4. Run: python app.py (or python3 app.py)
5. Follow the on-screen menu instructions.

Menu Options:
- Option 1: List all teams
- Option 2: Show player stats (enter "Tom Brady" or "Dak Prescott")
- Option 3: List games in NFL 2021 (enter "NFL" and "2021")

The database file "statsphere.db" will be created automatically on the first run.
